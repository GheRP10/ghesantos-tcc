14. CONCLUSÃO

A realização deste trabalho permitiu analisar, projetar e desenvolver um sistema de gestão de ordens de serviço voltado para uma loja de conserto de smartphones, evidenciando a importância da tecnologia da informação como ferramenta de apoio à organização e ao controle dos processos operacionais. A partir do levantamento dos fluxos de trabalho e das necessidades do negócio, foi possível compreender os principais desafios enfrentados nesse tipo de estabelecimento, especialmente no que se refere ao controle das informações, rastreabilidade dos serviços e acompanhamento financeiro.

Com base nessa análise, foi desenvolvido um protótipo de sistema utilizando a plataforma ASP.NET com linguagem C# e banco de dados relacional, contemplando funcionalidades essenciais como cadastro de clientes, produtos, ordens de serviço e formas de pagamento. A modelagem do sistema e a definição dos requisitos demonstraram como uma solução informatizada pode contribuir para a padronização dos processos e para a redução de falhas decorrentes de controles manuais.

Os resultados obtidos indicam que o sistema desenvolvido atende aos objetivos definidos neste trabalho, servindo como uma solução viável para apoiar a gestão de uma assistência técnica de smartphones. Ressalta-se que o sistema possui caráter acadêmico e foi desenvolvido e executado em ambiente local, como protótipo funcional, atendendo plenamente ao escopo proposto.

Dessa forma, conclui-se que o desenvolvimento de sistemas de informação voltados para a gestão de ordens de serviço constitui um recurso relevante para pequenas empresas do setor de manutenção de dispositivos móveis, contribuindo para a organização dos processo, controle das informações e apoio à gestão do negócio.
